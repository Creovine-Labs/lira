# Lira AI — Cinematic Product Launch Script

**Target Duration:** 90 seconds  
**Style:** Apple / Anthropic / Google — no voiceover, no talking heads  
**Audio:** Ambient electronic / orchestral build — starts quiet, peaks at feature reveal, resolves clean  
**Visuals:** Screen recordings from `/launch-demo`, text overlays, slow fades  
**Tone:** Calm, confident, inevitable

## POST-PRODUCTION NOTES

- **No voiceover.** The text overlays carry the message — clean, minimal
- **Transitions:** Cross-dissolve 0.4s between all scenes. No hard cuts except logo opening
- **Text style:** White, SF Pro or Inter, 28–34px, centered bottom, 70% opacity background pill — or pure white on dark moments
- **Aspect ratio:** 16:9 for YouTube / LinkedIn. Export a 9:16 crop for Instagram Reels / TikTok
- **Color grade:** Slight desaturation on the UI recordings. Warmer tone on any person-shots if added later
- **Logo intro:** Use the full spin animation — do NOT cut it short. The pause after the name appears is intentional
